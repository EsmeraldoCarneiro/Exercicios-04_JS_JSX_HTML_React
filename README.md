# 📑 Exercícios Práticos - ReactJS com Vite

Este repositório contém uma série de exercícios fundamentais desenvolvidos para praticar os conceitos de **Props**, **Componentes de Função** e **Componentes de Classe** utilizando o ecossistema React com Vite.

## 🚀 Tecnologias Utilizadas

* **ReactJS**: Biblioteca principal.
* **Vite**: Ferramenta de build e automação de ambiente.
* **JavaScript (ES6+)**: Lógica e manipulação de dados.
* **CSS-in-JS**: Estilização inline para componentes rápidos.

---

## 🛠️ Atividades Desenvolvidas

### 1. Componente de Função com Props

* **Objetivo**: Criar um componente de função `CompA` que recebe `nome` e `sobrenome`.
* **Conceito**: Desestruturação de Props em funções.
* **Resultado**: Exibição da mensagem: "Olá, Esmeraldo Junior".

### 2. Componente de Classe com Props

* **Objetivo**: Refatorar o exercício anterior utilizando a sintaxe de Classe.
* **Conceito**: Uso do `this.props` e herança de `React.Component`.

### 3. Reutilização de Componentes

* **Objetivo**: Utilizar o mesmo componente `CompA` quatro vezes no `App`.
* **Conceito**: Escalabilidade e o princípio DRY (Don't Repeat Yourself).
* **Entrada**: Lucas, Carla, Pedro e Beatriz.

### 4. Lógica Matemática com Props

* **Objetivo**: Componente de classe `Adicao` para calcular a soma de `x` e `y`.
* **Conceito**: Manipulação de dados numéricos dentro do método `render()`.
* **Resultado**: Cálculo de 7 + 23 = 30.

### 5. Componente Informativo (Saúde)

* **Objetivo**: Componente de classe `Vacina` para exibir informações sobre imunizantes.
* **Conceito**: Interpolação de strings e estilização temática.
* **Resultado**: "Coronavac é uma das vacinas contra a Covid-19."
